# Analysis

## Layer 11, Head 1

In both sentences it is clear that there's a pattern where each word pays attention to the word following it with hightened attention to verbs followed by prepositions such as "picked up" or "went out".

Example Sentences:
- Then I picked up a [MASK] from the table.
- So we went out and grabbed a [MASK].

## Layer 10, Head 3

Similar to the previous analysis, there is a very strong pattern where earch word pays attention to the word following it with the difference that attention is not scattered and is equally intense between condecutive words.

Example Sentences:
- Then I picked up a [MASK] from the table.
- So we went out and grabbed a [MASK].
